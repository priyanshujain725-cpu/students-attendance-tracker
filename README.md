# Student Attendance Tracker

## Project Overview
This is a simple Python program to mark and track student attendance. Users can mark a student's attendance for a specific date, save the data to a file, and view attendance records by student name. It is designed to be user-friendly and suitable for beginner programmers.

## Features
- Mark attendance by entering student name and date
- View attendance records displaying all dates a student was present
- Stores attendance data in a CSV format text file (`attendance.txt`)
- Simple menu-driven interface for ease of use

## How to Use
1. Run the script `attendance_tracker.py` in a Python environment.
2. Choose from the menu options:
   - Mark Attendance: Input student name and date to save attendance.
   - View Attendance: Enter student name to see attendance summary.
   - Exit: Close the program.
3. Attendance data is saved automatically to `attendance.txt`.

- Python 3.x installed on your system
